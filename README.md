## 搭建:
### production:
```
cd project_path
git checkout production
cd client
npm install
npm run build
cd ../server
npm install
node server.js
```
然后打开 `localhost:3000`


### dev:
```
cd project_path
git checkout master
cd server
npm install
PORT=5000 node server.js
#open a new terminal
cd project_path
git checkout master
cd client
npm install
npm start
```
然后打开 `localhost:3000`

## production和dev的代码区别：
- 遵循[redux-devtools](https://github.com/gaearon/redux-devtools)的指导。devToolsEnhancer在production里去掉。
即 [index.js](client/src/index.js) createStore不应该带有devToolsEnhancer

- [server.js](server/server.js)配置有所区别
